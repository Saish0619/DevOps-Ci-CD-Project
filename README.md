# DevOps CI/CD Pipeline + Docker Project

## Tools Used
- Git, GitHub
- GitHub Actions
- Docker
- Python Flask
- YAML Pipelines
- Linux Commands

## Pipeline Workflow
1. Code pushed to GitHub
2. GitHub Actions triggers automatically
3. Pipeline installs dependencies
4. Runs basic test server
5. Builds Docker image
6. Deployment done locally using Docker

## What This Project Shows
- CI/CD fundamentals
- Version control workflow
- Containerization
- Automation
